# Интерпретатор модельного языка программирования

Проект представляет собой консольный интерпретатор учебного модельного языка, реализованный на C++.

Интерпретатор выполняет следующие фазы:
- **Лексический анализ** – разбивает исходный текст на лексемы.
- **Синтаксический анализ** – методом рекурсивного спуска проверяет корректность структуры программы.
- **Семантический анализ** – контролирует контекстные условия (объявление переменных, совместимость типов).
- **Генерация внутреннего представления (ПОЛИЗ)** – переводит программу в постфиксную запись.
- **Интерпретация ПОЛИЗа** – выполняет программу, используя стек значений.

## Синтаксис языка
program {
<описания>
<операторы>
}

### Типы данных
- `int` – целые числа.
- `real` – вещественные числа.
- `string` – строки (в двойных кавычках).

### Операторы
- Присваивание: `x = выражение;`
- Ввод: `read(идентификатор);`
- Вывод: `write(выражение, ...);`
- Условный оператор: `if (условие) оператор else оператор`
- Цикл с предусловием: `while (условие) оператор`
- Цикл с постусловием: `do оператор while (условие);`
- Переход к следующей итерации: `continue;`
- Составной оператор: `{ операторы }`

### Выражения
Поддерживаются арифметические (`+`, `-`, `*`, `/`, унарный `-`), логические (`and`, `or`, `not`) операции и сравнения (`==`, `!=`, `<`, `>`, `<=`, `>=`). Приоритет операций стандартный.

Строки можно конкатенировать оператором `+`.

## Сборка и запуск
Требуется компилятор с поддержкой C++11 или новее.
``` bash
g++ -std=c++11 main.cpp scanner.cpp parser.cpp interpretator.cpp -o interpreter
./interpreter program.txt
```
###Пример программы
Файл program.txt:
``` C
program {
    int x, y = 5;
    real pi = 3.14;
    string s = "Hello";
    x = y + 10;
    write(x);
    write(pi);
    s = s + " world";
    write(s);
    if (x > 0 and y < 20)
        write("Condition true");
    else
        write("Condition false");
    do {
        x = x - 1;
        continue;
```
Ожидаемый вывод при вводе 42 на запрос read(x):
``` C
Синтаксический анализ завершен успешно.
15
3.14
Hello world
Condition true
42
0
```
## Структура проекта
type_of_lex.h – перечисление типов лексем и класс Lex.

ident.h – класс для хранения информации об идентификаторах.

scanner.h / scanner.cpp – лексический анализатор.

parser.h / parser.cpp – синтаксический и семантический анализатор, генератор ПОЛИЗа.

interpretator.h / interpretator.cpp – исполнитель ПОЛИЗа.

main.cpp – точка входа.

program.txt – пример программы на модельном языке.

## Особенности реализации
Метод рекурсивного спуска для синтаксического анализа.

Семантический контроль типов согласно таблице совместимости.

Генерация ПОЛИЗа с поддержкой условных и циклических конструкций.

Backpatching для оператора continue внутри do-while.

Интерпретатор использует стек значений с динамической типизацией.
